JS OBJECTS

  1. In JavaScript, almost "everything" is an object.
  2. Booleans can be objects (if defined with the new keyword)
  3. Numbers can be objects (if defined with the new keyword)
  4. Strings can be objects (if defined with the new keyword)
  5. Dates are always objects
  6. Maths are always objects
  7. Regular expressions are always objects
  8. Arrays are always objects
  9. Functions are always objects
  10. Objects are always objects

    11. All JavaScript values, except primitives, are objects.

    JavaScript Primitives

    A primitive value is a value that has no properties or methods.

    A primitive data type is data that has a primitive value.

    JavaScript defines 5 types of primitive data types:

    1. string
    2. number
    3. boolean
    4. null
    5. undefined

      6. Primitive values are immutable (they are hardcoded and therefore cannot be changed).
